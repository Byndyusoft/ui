## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import Highlighter from '@byndyusoft-ui/highlighter';" />

2. Call the component and give it the desired props, for example:

-   Simple example

<Canvas
    sourceState="shown"
    of={HighlighterStories.BaseStory}
    story={{ height: '50px' }}
/>

-   Highlight with overlapping search values

<Canvas
    sourceState="shown"
    of={HighlighterStories.HighlightWithOverlappingSearchValuesStory}
    story={{ height: '50px' }}
/>

-   With custom highlight

<Canvas
    sourceState="shown"
    of={HighlighterStories.CustomHighlightStory}
    story={{ height: '50px' }}
/>

-   Interactive with ignore spaces

<Canvas
    sourceState="shown"
    of={HighlighterStories.InteractiveWithIgnoreSpacesStory}
    story={{ height: '100px' }}
/>
